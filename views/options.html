Синхронизация

Проверка статуса...

Данные канбан-доски будут синхронизироваться между устройствами через ваш Google Drive. Синхронизация c Google Drive работает только в Chrome.

Для подключения Яндекс.Диска зарегистрируйте приложение на oauth.yandex.ru:

  1. Перейдите на oauth.yandex.ru
  2. Создайте приложение → тип «Веб-сервис»
  3. Добавьте права cloud_api:disk.write и cloud_api:disk.read
  4. Скопируйте ClientID и вставьте ниже, нажмите «Сохранить»

Получение токена:
1. Нажмите кнопку выше — откроется страница авторизации Яндекса
2. Разрешите доступ — Яндекс перенаправит на https://oauth.yandex.ru/verification_code
3. Скопируйте access_token из адресной строки (после access_token=) и вставьте ниже
4. Нажмите «Сохранить токен»
